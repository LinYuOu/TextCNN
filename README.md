# TextCNN
一个文本分类的项目，数据集是中文医学数据，三万多条训练数据，七千多条测试数据
将句子进行词嵌入后构造输入矩阵，经过卷积后再加入位置信息，最后通过线性层进行分类
-	对中文医疗数据进行词嵌入，将词向量构造成类似图像的数据矩阵
-	使用CNN对输入的矩阵提取特征，输出特征矩阵
-	将位置信息融入特征矩阵
-	最终在测试集上达到80%的准确率。
